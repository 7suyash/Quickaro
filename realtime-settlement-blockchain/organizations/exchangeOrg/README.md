# Exchange Organization

This directory contains the MSP (Membership Service Provider) crypto materials for the **Exchange Organization** in the settlement network.

## Contents

Generated by `cryptogen` when running `start-network.sh`:

- `msp/` — Organization-level MSP (root CA certs, TLS CA certs)
- `peers/` — Peer node identities and TLS certificates
- `users/` — Admin and client user identities
- `ca/` — Certificate Authority server configuration

## Role

The Exchange Organization submits trade records to the settlement smart contract and endorses settlement transactions.
